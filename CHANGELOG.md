Changelog
==========

### 1.5.0 - October 21st, 2025

- Added admin settings page for configurable tracking options
- Added option to track internal clicks (adds `track-internal="true"` attribute)
- Added option to track outbound clicks (adds `track-outbound="true"` attribute)
- Added option to track customers with UUID (adds `track-customers="true"` attribute)
- Script tag now always includes `void-analytics-tag="true"` attribute
- Updated plugin name to "Void Analytics Integration"
- Updated minimum PHP requirement to 7.2.0
- Improved plugin description

### 1.4.0 - November 14th, 2024

- Initial integration script
